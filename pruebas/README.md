# Carpeta de pruebas

Esta carpeta contiene la base de pruebas para validar los objetivos del MVP definidos en el `README.md` principal del proyecto.

## Contenido
- `plan-de-pruebas.md`: casos de prueba iniciales para autenticación, productos, inventario, ventas y reportes.

## Uso sugerido
1. Ejecutar los casos de prueba por prioridad.
2. Registrar evidencia por caso.
3. Marcar bloqueos y convertir casos críticos a pruebas automatizadas.
