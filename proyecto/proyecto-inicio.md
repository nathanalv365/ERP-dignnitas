# Proyecto: Inicio

Este archivo marca el arranque del proyecto ERP-dignnitas con una definición mínima de alcance.

## Alcance inicial
- Módulo de autenticación
- Módulo de productos
- Módulo de inventario
- Módulo de ventas
- Módulo de reportes básicos

## Entregables de la fase 1
1. Modelo de datos inicial.
2. API base para productos e inventario.
3. Flujo de venta con actualización de stock.
