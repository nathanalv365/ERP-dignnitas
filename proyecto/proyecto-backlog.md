# Proyecto: Backlog inicial

## Historias priorizadas
1. Como administrador, quiero crear y editar productos para mantener el catálogo.
2. Como operador, quiero registrar entradas y salidas de inventario para controlar stock.
3. Como operador, quiero registrar ventas para llevar trazabilidad comercial.
4. Como administrador, quiero ver ventas por fecha para analizar resultados.

## Definición de terminado (DoD) mínima
- Historia implementada y probada.
- Sin errores críticos en flujo principal.
- Registro básico en documentación técnica.
